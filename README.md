CopyFile - Программа копирования файлов с отображением прогресса
Простая утилита для копирования файлов с отображением прогресса в реальном времени, написанная на Go.

*Базовый синтаксис*
```bash
copyfile <исходный_файл> <целевой_файл>
```

**Аргументы**
- <исходный_файл>	Путь к файлу, который нужно скопировать	Да
- <целевой_файл>	Путь, куда будет скопирован файл
